Instructions for running back end:
	1) Go to the Back End directory in command line
	2) Type the following commands:
		>>python
			>>from crawler import crawler
			>>crawler = crawler(None,'urls.txt')
			>>crawler.crawl()
			>>inverted_index = crawler.get_inverted_index() # For getting the inverted index
			>>print inverted_index # For printing the inverted_index
			>>resolved_inverted_index = crawler.get_resolved_inverted_index() # For getting the resolved inverted index
			>>print resolved_inverted_index # For printing the resolved_inverted_index
